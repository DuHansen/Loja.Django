# Loja.Django
Projeto Store

No Prompt de comando digite,

( cd\ )

para ficar na pasta raiz, depois instale VIRTUALENV,

( pip install virtualenv )

Depois,

( virtualenv store )

Ativando,

( scripts\ activate.bat )

Instale PILLOW,

( pip install pillow)

Inicializar projeto,

( django-admin startproject storeweb )
